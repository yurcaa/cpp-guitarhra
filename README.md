# Guitar Hra

Rhythm game I made in C++ with SFML. Three lanes, blocks fall down, hit the right key when they reach the bottom.

![gameplay](screenshot.png)

## Controls

| Key | Action |
|-----|--------|
| `A` | Left lane (green) |
| `S` | Middle lane (red) |
| `D` | Right lane (blue) |
| `F11` | Fullscreen |
| `Escape` | Exit fullscreen / close |
| `R` | Restart |

Download the latest release, extract the zip and run `GuitarHra.exe`.
